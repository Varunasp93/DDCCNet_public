# DDCCNet_public
Public version of DDCCNet code. This contains files to train and test DDCCNet_v1, v2 and v3
In each of these folders there are python scripts to,
1. Generate Data for Training
2. Training the models
3. Data generation for testing
4. Testing the models

|Function                  |DDCCNet_v1            |DDCCNet_v2           |DDCCNet_v3           |
|--------------------------|----------------------|---------------------|---------------------|
|Generate Data for Training |Data_Generation_v1.py|Data_Generation_v2.py|Data_Generation_v3.py|
|Training the models        |Train_NN_v1.py       |Train_NN_v2.py       |Train_NN_v3.py       |
|Data generation for testing|Test_NN_v1.py        |Save_test_feats_v2.py|Test_NN_v3.py        |
|Testing the models         |Test_NN_v1.py        |Test_NN_v2           |Test_NN_v3.py        |
